# 10-Phase Implementation Plan

| Phase | Task | Description |
|-------|------|-------------|
| 1 | Environment Setup | Install Python/Node, create virtual env, and folder structure. |
| 2 | Dataset Loading | Ingest UNSW-NB15 and perform initial Exploratory Data Analysis (EDA). |
| 3 | Data Cleaning | Implement the preprocessing pipeline (encoding, scaling, null handling). |
| 4 | Feature Engineering | Select top predictive features and handle class imbalance. |
| 5 | Model Training | Train Random Forest (Supervised) and Isolation Forest (Unsupervised). |
| 6 | Model Evaluation | Generate classification reports and confusion matrices. |
| 7 | Threat Detection Logic | Create the alert generation system with severity mapping. |
| 8 | Visualization | Build graphs for feature importance and anomaly scores. |
| 9 | GitHub Publishing | Initialize repository, write commits, and document README. |
| 10 | Final Output & Proof | Capture screenshots, record demo, and update portfolio. |
